# Taller Los Compadres SPA - version 1

Web estatica profesional para Taller Los Compadres SPA.

## Ruta

Abrir `index.html` directamente o levantar un servidor local dentro de esta carpeta.

## Archivos principales

- `index.html`: estructura de la pagina.
- `styles.css`: estilos responsive.
- `script.js`: menu mobile, estado del header y enlace dinamico de WhatsApp.
- `brief-marca.md`: investigacion y decisiones de marca.
- `assets/`: logo, flyers enviados y hero generado para la web.

## Assets

Todos los assets estan en una sola carpeta `assets`, sin subcarpetas.

## Investigacion

Se uso el enlace de Google Maps enviado por el usuario, las imagenes promocionales adjuntas y una busqueda web publica para corroborar el contexto visible del taller. No se inicio sesion en Instagram ni Google.
